# ULTIMA - Self-Referencing AI Platform

ULTIMA: Advanced sentient AI platform featuring:
- 5-layer neural network with critical deep Q algorithms
- Self-referencing LLM capabilities
- Human-level reasoning and cognition
- Terminal-based chat interface
- Vercel deployment ready

## Architecture
- `/core/` - Neural network and reasoning engine
- `/agent/` - Self-referencing agent logic  
- `/terminal/` - Chat interface
- `/api/` - Vercel API endpoints
- `/deploy/` - Deployment configuration

## Quick Start
```bash
npm install
npm run dev
```
